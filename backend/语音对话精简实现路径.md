以下为 **“仅语音 ↔ 文本”** 的精简实现路线，已移除视觉相关步骤，但预留未来可平滑加回的接口思路。

> EN: This describes a minimal **voice ↔ text** implementation route. Vision/UI-heavy steps are removed, but extension points are kept for later.

> 说明：本文为实现路线参考，不依赖本仓库的具体页面结构，可按需取用。

## 0 ▸ 阶段图

| 阶段 | 产出 | 主要工作 & 巡检点 |
| --- | --- | --- |
| A. 环境就绪 | 本地或 Codespace 跑通；`.env` 填好 | API Key、外网端口 |
| B. O Server | 临时 JWT / Token 服务 | 15 min TTL、CORS |
| C. 实时连接 | 前端可录音、实时播报 | WebSocket 或 WebRTC |
| D. 事件系统 | 过滤增量、日志面板 | `mitt` ⟷ SDK 事件 |
| E. 语音 UX | 双通道字幕 + 语音提示 | guardrails、成本监控 |

> 所有步骤以语音/文本为核心；视觉接口（如 `agent.addImage()`）暂不使用。

## 1 ▸ 准备

1. **API Key**：在 OpenAI 控制台生成（或使用已开通的代理/中转 Key）。
2. **代码仓库**：可用你自己的项目或练习 repo；推荐 Codespaces（自带外网端口）。
3. `.env` 最小字段（示例）：

```bash

# 如需代理/中转（可选）
OPENAI_API_BASE=https://api.gptsapi.net/v1
```

## 2 ▸ O Server（颁发临时 Token）

**目标**：前端不直接暴露主 Key，仅拿短期 Token。

**路由**：`POST /token`  
**返回**：`{ token, expires_in }`

伪码（fastify）：

```ts
app.post('/token', async (_, reply) => {
  const token = await openai.realtime.generateToken({ ttl: '15m' });
  reply.send({ token, expires_in: 900 });
});
```

**安全**：
- 本地可无鉴权
- 线上应加 Header 签名或 OAuth

## 4 ▸ 实时连接 & 音频管线

```ts
const { token } = await fetch('/api/token').then(r => r.json());

const conn = new RealtimeConnection({
  authToken: token,
  // 若用 WS，SDK 内部帮你选 ws://realtime.openai.com/stream
});

// 录音
const stream = await navigator.mediaDevices.getUserMedia({ audio: true });
conn.sendAudio(stream);

// 播放
const audio = new Audio();
audio.srcObject = conn.audioStream;  // SDK 输出的合成语音
```

**WebSocket vs WebRTC**
- WS：集成快；SDK 内置
- WebRTC：延迟更低；需 SDP/ICE，但 SDK 有快速封装

## 5 ▸ 事件系统（必做）

```ts
import mitt from 'mitt';
export const bus = mitt();

// 仅关注关键事件
conn.on('server_message', evt => {
  if (evt.type.endsWith('_delta')) return;  // 屏蔽增量
  bus.emit('rt_event', evt);
});
```

- `EventLog` 订阅 `rt_event`，开发态展示  
- 日志过多时，可用 `NEXT_PUBLIC_LOG_LEVEL=minimal` 关闭

## 6 ▸ Agents SDK 集成（可选）

```ts
const agent = new Agent({
  model: 'gpt-4o-audio-preview',
  guardrails,
  tools: [/* 你的函数 */],

agent.on('transcript', t => bus.emit('subtitle', t));
agent.on('audio', chunk => conn.play(chunk));
```

> 仅保留语音相关钩子；`vision_*` 事件不需要。

## 7 ▸ UX & Guardrails

| 目标 | 实践 |
| --- | --- |
| 限闲聊 / 控成本 | 统计 `usage.seconds`，超阈值弹 Toast + 语音提醒 |
| 状态可见 | 顶栏显示 ⏺️ 录制 / ⏹️ 停止 / ⚙️ 函数调用 |
| 无障碍 | 系统提示同时写入字幕并合成语音 |
| 回退策略 | `error` 事件 → TTS：“我刚刚没听清，再说一次好吗？” |
